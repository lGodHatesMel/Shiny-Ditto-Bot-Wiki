# BREADING DITTO

**Commands**
`!swshdittotrade` or `!swshdt` or `!swshditto`

## How To USe

!swshittotrade <optional code> <stat modifiers> <language> <nature>Â - Will initiate a trade for a Ditto with the requested parameters.

### Language modifiers: 
Enter language name as per ALM's extra Showdown set input format.
